日本见闻

发表于 2018-08-09   |   分类于 写作   |     |   阅读次数   |   本文总阅读量 次

2018年 8月 8日 星期三 16时16分27秒 CST
Weather in Beijing => 28 °C - UVI => 9.68 - Wind => 4 m/s SSE - Humidity => 78 % - Pressure => 1007 hPa - Sunrise => Aug 08 05:19:25 AM - Sunset => Aug 08 07:20:08 PM

在不算百忙的时间中抽出了7天，陪着老婆，孩子，还有朋友，一行8人，去了趟日本，并小范围的把大阪，京都，奈良的相关景点粗略的过了一遍。当然，我的角色，更多的是陪同、摄影、苦力。其实，都还好了，这次去日本比我预想的要好很多。总是频频，主动和老婆说，日本签证时间办短了。应该来个三年，或是五年签证。这样下次去就方便了。是的，日本对于我来说，真的是那种去一次不够的国家。抛开政治和历史问题不谈，在交通，环境，人文，尤其是日本人的服务意识比我去过的任何一个国家都要亲切，都要热情。而且，日本人一丝不苟的做事态度，真的让我记忆犹新，印象深刻。。看不出一丝国内的浮躁和抱怨，大家都勤勤恳恳，规规矩矩，不带半点虚假。

点头哈腰

是的，只要是在任何交易场合。也就是你在日本，任意大的，小的店面。以钱换物的时候，换任何物品。对方在接过钱后，都会对你“点头哈腰”，嘴里说着“#$$%#^#&”我听不懂的日语。那肯定是感谢，欢迎下次光临之类的话语。每当这时，我也会以类似的，幅度较小的动作，带以微笑来回敬他们。而每当这时，你都会感觉在日本花钱，你得到了应有的上帝的感觉。而这种感觉，在国内，已经很久，很久没有体会过了。除了个别服务较好的商铺，如：海底捞等。而国内的一些公家单位的服务更是让人哭笑不得。付钱的人都感觉自己像个孙子，收钱的人是大爷。这样是不是感觉有些怪怪的？

浮躁

在国内，浮躁，随处可见。职场，生意场，股市，楼市，影视圈，甚至整个社会都弥漫着浮躁的气息，就像整个夏天到处充斥的热浪。让人难受，让人皱眉，快要窒息的感觉，一分钟也不想呆在室外。可能浮躁的背后与钱的关系太紧密了吧。可能钱是浮躁背后最大的幕后黑手吧。可能整个社会都在一切朝钱看吧。一切朝“钱”看的后果，可想而知。破坏了环境，亲情，空气，甚至整个社会的风气和氛围。是啊，上面说到的这些，我在日本真的没有看见。环境，空气，城市规划，治理，甚至是足球水平比我们要高出N个档次。是的，有的时候，可以从侧面现象了解事物的本质。就像孩子的外在表现，可以看出孩子的父母是如何教育孩子的。父母在家经常骂人，那么孩子必定会骂人。孩子会骂人，那么父母在家一定经常骂人。车轱辘话来回说。道理就是这么的简单。

在日本，贫富差距没有那么大，所以不用为钱而大为浮躁，农民的工资比公务员还高；反过来想想中国的贫富差距，我们也就不多说了。

日本人的房子意识，年轻人工作三五年后，贷款买房子，父母也不资助的，自己贷个30多年的款，每月老老实实地拿出工资的将近一半用于还贷，在一家公司老老实实地上班多年。之后升职加薪，手头宽裕了，也没有想过再买一处房子投资。他们说那是房地产公司干的事。人生拥有一处作为家的房子已经足够了，比房子更值得追求的事儿还有许许多多。

在日本，有学区，没有学区房，你可以向政府申请任何正当理由，方便你子女上学的学校去上学。

  1. 如孩子有疾病需要离医院近，那么你就可以在离医院近的学校上学。
  2. 如离异，需要爷爷辈照顾，那么你就可以申请在离爷爷辈家近的学校上学。
  3. 还可以日本国内留学，如果在城市里，你觉得孩子天天在家玩游戏，很多耽误孩子，你可以申请，让孩子去到偏远山区，或是小岛的学区去读书，寄宿在学校，或是附近的家庭里。城里的小孩到农村生活、学习，他们看到了与城里截然不同的风景，感受大自然的美妙。体验农村的小学，结识了更多山村的小朋友。即使是小学生也自己动手做家务，培养了独立自主的意识。
  4. 而日本的学校教育资源都非常平均，单就校舍面积来说：关于校舍的面积，学生人数40人以下的，必须达到500平方米。41人到480人，则按照500＋5×（学生人数－40）这个公式计算该有的面积，481人以上的，按照2700＋3×（学生人数－480）的公式来计算。学生240人，运动场面积必须达到2400平方米，大约一人10平方米。在山区或者小岛上，有的小学只有数十名以至数名学生，但是学校的环境设施也毫不含糊，操场、体育馆、游泳池等等该有的都有。

在日本，环境非常好，真的是特别干净，机动车都跟新的似的，感觉街上的车全部都是刚刚从4S店提回家一样。就连自行车都非常的干净。这是怎么做到的，让我百思不得其解。

医疗就更不用说了吧，日本出色的医疗体制，不仅用心呵护着全体国民的生命质量，也为日本成为世界最长寿国家（据2017年统计，平均寿命女性87.14，男性80.98）做出了巨大的贡献。在日本，我也很少看见肥胖的人。

说了这么多，大家不要怪我崇洋媚外，我说的是事实，而在这事实下，受益的是日本本国民，与我也没有半毛钱关系。我也只是把自己的所见所闻如实的说出来而已。

尊重

其实，这些东西已经涵盖了收入，住房，教育，医疗，环境。而这些东西就是与老百姓息息相关的最根本，最重要的东西。也是老百姓最关心的问题。如果这个国家把这些事情都处理的比较得当，我想，我们的国家也会是一个非常受人欢迎和尊重的国家。

是的，尊重，在今年世界杯的赛场上，日本队和日本球迷已经得到了全世界的尊重。试想一下，如果中国队进军俄罗斯会做到同样的效果吗？值得我们思考。

为什么日本，日本队，日本球迷可以做到这样？可以赢得旅行者的亲睐，硬实力提升，受人尊重？我不知道，我想，通过学习和研究日本的著作，日本的历史，日本的影视剧（，更甚者让我有种学习日语的冲动）。是可以得出结论的，也许网络上已经有答案，只是我现在不知道，不够深入。这也是我决定从今天开始每天写一篇深度思考的文章。做一个能够深度思考的人。做一个长期积累。有自己产品或作品的人。

下面是我的几幅摄影作品，在后面，我也会不定期的写出我对摄影，摄像，旅行，英语学习，身体锻炼等方面的一些感悟和心得，资源共享，同时也鞭策我自己不断前行。

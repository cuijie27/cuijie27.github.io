我做了这么多，为什么还是得不到我想要的？

发表于 2018-08-17   |   分类于 思考   |     |   阅读次数   |   本文总阅读量 次

应该有包括我在内为数不少的人都有这样的困惑，这篇文章只是想表达一下我对这个问题的看法，并不一定可以解决问题。或是随着年龄，能力，阅历的增长，上面的这个问题也就不再是个问题了。

1. 方向是否正确

首先，得确定好自己都做了哪些事情，即方向是否正确。方向不正确，做得越多，离目标越远。那无异于南辕北辙，最后只能是竹篮打水一场空。比如：

我要达成目标E，而E目标达成的前提条件是做好A，B，C，D。
但，偏偏我做的事情，可能就是差那么一点。如：我只是做好了A，B，C，而没有做好D，那么要达到目标E肯定是有难度的。或是我干脆就没有做A，B，C，D而是去做X，Y，Z。这完全与目标背道而驰，是我们坚决不应该去做的。

2. 是否能找到方法

其实，上面由A+B+C+D最后导出E的这个过程，很简单，只是一个简单的充分必要条件推理。我们在现在的工作，生活，学习中大部分都是以目标为导向，只要是能找到目标需要的几个充分条件，我们就可以轻松完成我们要达成的目标。

这是在我们能找到A，B，C，D的前提下，
但关键问题是我们找不到A，B，C，D呢？
又或是我们找到其中的A，B，C，但是我们牙根儿不知道D也是必要条件。如果是这样，那就彻底尴尬了。

所以，正确分析问题的能力是非常重要的。至于如何分析问题，那又是另一篇文章的事情了。

3. 积累时间是否足够长

举一个最简单的例子。有常识的人都知道，孕妇的怀孕周期是40周，不管你娶多少个老婆为你生孩子，也只能是40周的时间，周期不可缩短，自然规律无法打破。

《异类》这本书中提到的“10000小时定律”——想要出类拔萃，就要努力至少10000小时。

其实，技能的熟练程度与练习的时间长短有关，卖油翁的“无他，但手熟尔。”、罗马不是一天建成的、冰冻三尺，非一日之寒。这些都可以说明问题。

时间是个非常奇妙的东西，很多事情都可以因它而发酵，而且会得到长足的发展。好的事情会随着它越来越好，如：技能的学习、有潜力的股票、健身等等。坏的事情会随着它越来越坏，如：癌症……

时间的重要性，我们不能小视。

4. 是否充分预估难易程度

比如：“我想练成道恩·强森(Dwayne Johnson)那一身完美的肌肉，可以吗？”

当然可以，每个人都有梦想，我们无法干预，也没有道理阻拦。

但，人贵自知，我们总得需要知道自己的现状，自己可以运用的资源和时间是否能够帮助自己达到那样一个目标。

  1. 你现在的年龄是否偏大？
  2. 你现在的体重是否合适？
  3. 你是否能够保证持续，稳定，大量的时间、精力的投入？
  4. 你是否可以坚持下来？
  5. 你是否……。
  6. 这些都是未知数。

最好的情况应该是，我们先实现一个容易达到的小目标（就像我们的建林大大说的那样，先完成一个亿的小目标），然后再一步一步向更高层次进发。循序渐进。这是正确的思路。切忌急功近利。

5. 目标是否可行

我们在做一件事情之前，一定要想清楚这件事儿。目标无法实现的事情，或是在我们有限的寿命内无法完成的目标，就应该果断放弃。比如：我想研制长生不老的药丸。或是其它一些比较虚无缥缈的东西。

而应该是把现有已知非常成熟的知识，理论，最好是成果摆出来，再把自己的目标拿出来。看看前后两者是否存在一条可能的通路。

就像区块链世界中的Mixin(我非常看好的区块链项目之一)。其中运用到的所有计算机技术，均为业界中非常成熟，并已经验证过的技术，在此基础上，Mixin所倡导的所有特点都是可以实现的。这就非常的靠谱了。

6. 结论

说了这么多，也都只是纸上谈兵，真正关键还在平时的每分，每秒。牢记使命，不忘初心不是句空话。把目标设置得现实可行，一直执拗的和自己较劲下去。终会有所收获。

始终以100分的标准来要求自己，即使最后尽全力后，得了80分，不也是值得高兴的事情吗？

日本，一个值得思考的国家。

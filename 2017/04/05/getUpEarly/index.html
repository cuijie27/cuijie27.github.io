写给想要早起的亲们

发表于 2017-04-05   |   分类于 升级   |     |   阅读次数   |   本文总阅读量 次

今天是我连续早起的第174天，这个数字相当于接近6个月的日子里我都早起了。

平均算下来，每次早起比不早起的时候要多利用1.5小时。所以这段早起的日子里，我要多利用了(174*1.5=)261小时，近11天的时间。要知道，这近11天的早起多出来的时间是没有任何干扰的，所有的人和事都不会在这个时候打扰你。这也是当初我为什么要早起的原因之一——在一大块安静的时间内专心投入精力去做一件事情。

在别人还在被窝熟睡的时候，我利用这多出来的时间写文字。到现在为止，每天码字已经成为我持续做的最好的事情之一。原因很简单，早起为每天码字提供了时间保证。也就是说， 只要能持续每天早起，我就能持续每天码字 。一举多得。而码字的好处大家都懂的，在这里就不展开了。

每天当我在上班路上，看见熙熙攘攘的人流和车流奔波在上班途中；当我在公司早已进入工作状态，而我的同事们拿着早点姗姗来迟时，我的心理是踏实的，自信的，自豪的。因为早起，我早已经计划和完成了一天当中很多重要的任务和工作。放眼望去，在早起这件事情上，我已经甩开了90%的人。而长时间持续的早起，我也可以毫不谦虚的说，甚至已经甩开了99%的人。早起领先后，在这一天的工作，学习，生活中就会步步领先，长时间持续下去就会演变为全面领先。

同时，在每个周末或假期，早起完成自己每天的任务后，甚至还可以带动家人趁早出游。错开拥堵的出行高峰；避开人满为患的景点；在饭店门口的排号长龙形成之前，我们就已经坐在里面享受美食。全天下来你会感觉到周末的美好时光变长了。是啊，早起这样的好习惯，让你在玩儿这件事情上都能占尽先机，玩儿出更多的时间。多出来的时间既要好好的成长，也要好好的玩儿，这才是获得幸福感的根本源泉所在。你看，早起可以直接增加幸福感，这样的好事为什么不做呢？应该做，而且还要持续，永久的去做。

另一点，早起必然导致要早睡。对成人来说最健康的睡眠时间是8小时；有的人可以做到每天睡7小时，甚至是6小时，而身体也能够适应得了。下表来自How Much Sleep Do We Really Need?

Average Sleep Needs by Age

由于人与人之间的差异，能保证大多数成年人健康的情况下的合理睡眠是7到9个小时。在5点起床的前提下，

  1. 如果我们只需要7个小时睡眠。那么我们上床入睡的时间是22点。
  2. 如果需要8个小时睡眠，上床入睡的时间是21点。
  3. 需要9个小时的睡眠，上床入睡的时间是19点。
  4. 需要6个小时睡眠的人，上床入睡的时间是23点。(研究证明，由于基因问题，这种只需要6小时睡眠的人是存在的。大概占人口比例的3%。不知道中国人的比例占多少？哈)

所以，长期来看，只需要少量睡眠时间就能保障健康状态的人，能获得更多的时间去学习，进步，成长。至于平时我们为什么早起起不来，原因其实很简单，我们睡得太晚了。如何早起？只需要睡得早一点，就可以早起了。

就像去到另一个时区的国家后，需要倒时差一样。如果一旦开始早起，改变之前旧的，不合理的作息时间将是解决早起问题的关键所在。而头一个星期，是改变老旧作息时间的关键期。在这期间一定要硬着头皮早上5点起床，慢慢的你会发现到了晚上9，10点钟，困意准时到来，这个时候只想去睡觉。因为你早上起得太早，为了补足这7，8个小时的健康睡眠时间，你的生物钟在此时会自动启动，促进大脑里的松果体释放褪黑素，褪黑素对睡眠的产生和维持都起着重要作用。就这样，慢慢的，早睡早起的好习惯将不再是别人的了。

与此同时，早睡早起的好习惯养成后，随之而去的将是一些不好的习惯。如睡前刷朋友圈，看无聊电视剧，综艺节目，等等。因为早起之后，你真的没有多余的精力去做这些事情了。

并且，只要开始早起，就根本停不下来。尤其是在微博上每天早起打卡。那一天天增长的早起的次数，是你早起很困难时的坚强的支柱，是一种信念，也是一种态度。不论是周末，小长假，黄金周，甚至是大年三十的早上5点。不论是在外地出差，还是在国外旅游。不论你是学生，是年轻的单身白领，还是有家庭有孩子的成年人，甚至是上岁数的老人。任何时间，任何地点，任何人都可以让早起一直伴随着自己，成为管理自己，把握时间的一种有利的工具。

The Early Bird Gets the Worm. 是的，这个老生常谈的观念，在西方也是通用的。bing 搜索 get up early benefits，你会看到很多西方世界对早睡早起的评价。

其实，在早起这件事情上，让我感触最深的是：

  • 我不再和以往喜欢睡懒觉的自己一样，被动的等待时间毫无意义的流逝。
  • 而是主动出击，主动去管理自己，让自己在时间面前主动起来。
  • 因为我们不可能管理时间，不可能主宰时间，不可能做时间的主人。
  • 但也不能任凭时间牵着自己的鼻子走，而我们能做的是：用大脑控制自己与时间友好相处。
  • 你若对时间倾尽所有，那么时间定不会亏待于你。因为你们已经是朋友了。复利效应最能说明这一点。

而早起则是管理自己和时间友好相处最直接的体现。

作家郑渊洁，篮球巨星科比都是riser。比你牛的人，还比你起的早。不早起的我们好像有点说不过去了吧。
